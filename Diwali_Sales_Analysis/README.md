#  Diwali Sales Analysis

     Overview  
This project analyzes customer purchasing behavior during the Diwali festival season using Python and Jupyter Notebook.  
The goal is to identify which customer segments contribute the most to sales and which product categories dominate during the festive period.

---

   Dataset  
The raw CSV file contains information such as:  
- Customer demographics  
- Marital status  
- State  
- Occupation  
- Age group  
- Product categories  
- Purchase amount  

---

   Tools Used  
| Tool | Purpose |
|------|---------|
| Python | Data manipulation & analysis |
| Pandas | Data cleaning & filtering |
| NumPy | Numerical computation |
| Matplotlib | Data visualization |
| Seaborn | Statistical charting |
| Jupyter Notebook | Code execution & documentation |

---

   Key Insights

- **Married Women** spent the most during Diwali  
- Top purchasing states: **Uttar Pradesh, Maharashtra, Karnataka**  
- Dominant age group: **26–35**  
- Leading occupations: **IT Sector, Healthcare, Aviation**  
- Most purchased product categories:  
  - Food items  
  - Clothing & Apparel  
  - Electronic gadgets

   Conclusion  
Married women aged 26–35, working in IT/Healthcare/Aviation sectors, make up the **highest-spending demographic**, particularly in the major states of **UP, Maharashtra, and Karnataka** — with strong interest in **food items, clothing, and electronics**.
